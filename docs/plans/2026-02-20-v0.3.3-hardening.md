**Step 3: Implement `readStoredDimensions` in `src/db.js`**

Add this function and export it. Opens the database read-only (no sqlite-vec needed), reads the `audrey_config` table, closes immediately.

```js
export function readStoredDimensions(dataDir) {
  const dbPath = join(dataDir, 'audrey.db');
  if (!existsSync(dbPath)) return null;
  const db = new Database(dbPath, { readonly: true });
  try {
    const row = db.prepare("SELECT value FROM audrey_config WHERE key = 'dimensions'").get();
    return row ? parseInt(row.value, 10) : null;
  } catch {
    return null;
  } finally {
    db.close();
  }
}
```

Add `import { existsSync } from 'node:fs';` at the top of `src/db.js` (it already imports `mkdirSync` from `node:fs`).

### Task 5: Isolate errors in encodeBatch validation
